# 🐾 System Obsługi Weterynarii

Nowoczesna aplikacja webowa do zarządzania wizytami i dokumentacją w placówkach weterynaryjnych.

## 📌 Technologie

- **Frontend:** Angular
- **Backend:** Java (Spring Boot)
- **Baza danych:** PostgreSQL
- **Usługi chmurowe:** Microsoft Azure (usługi AI)

## 👥 Role użytkowników

### 👤 Klient
- Rejestracja i logowanie
- Przeglądanie dostępnych weterynarzy
- Umawianie wizyt
- Podgląd historii leczenia zwierząt
- Powiadomienia o wizytach

### 🩺 Weterynarz
- Kalendarz wizyt
- Przegląd historii leczenia pacjentów
- Dodawanie wpisów medycznych
- Zarządzanie danymi zwierząt
- Komunikacja z klientami

## ⚙️ Główne funkcjonalności

- 🔐 Autoryzacja i rejestracja użytkowników (JWT)
- 📅 Zarządzanie wizytami
- 📋 Historia leczenia zwierząt
- 🔔 System powiadomień
- 🤖 Integracja z usługami Azure AI (np. rozpoznawanie mowy, analiza tekstu)
- 🧠 Wykorzystanie inteligentnych modeli do analizy danych medycznych

## ☁️ Azure AI

System wykorzystuje wyłącznie **inteligentne usługi Azure**, takie jak:

- **Azure Cognitive Services** – analiza języka naturalnego, ekstrakcja informacji z opisów medycznych
- **Azure Speech to Text** – możliwość zapisu wizyt głosowo
- **Azure Form Recognizer** – odczyt danych z formularzy lub wyników badań
- **Azure OpenAI** – generowanie podsumowań wizyt lub sugestii leczenia (opcjonalnie)
